COMBOOT

Syslinux поддерживает небольшие отдельные программы, используя формат
файлов, аналогичный
DOS'овским "*.com" файлам. Их 32-битная версия называется COM32 и тоже
поддерживается.
Простой API даёт доступ к ограниченному набору функций файловой системы
и консоли.

Подробнее см. comboot.txt (техническая информация).
